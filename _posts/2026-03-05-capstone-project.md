## Arduino Code & Sensitivity Calibration  

The Arduino code is designed to manage and control the various functions of the smart stool. It integrates sensor readings and processes the input to ensure accurate data display and response. The SENSITIVITY setting is configured to 80, which allows for a balanced sensitivity in detecting weight changes without being overly responsive to minor fluctuations. This ensures reliable operation during its usage.
