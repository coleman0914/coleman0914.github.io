---\ntitle: Capstone Project - The Smart Stool\nlayout: post\n---\n\n## Overview\n\nThe Smart Stool is a project that aims to innovate the way we interact with our seating arrangements by introducing various features that improve comfort and connectivity.\n\n## Features\n- **Adjustable Height**: A mechanism that allows users to change the height of the stool based on their preference.\n- **Smart Sensors**: Equipped with pressure and temperature sensors to monitor user experience.\n- **Connection to Mobile App**: Users can control settings through an integrated mobile application.\n\n## Technology Stack\n\n- Microcontrollers: Arduino or Raspberry Pi for core functionality.\n- Sensors: Various types for the necessary user inputs.\n- Mobile Development: Technologies such as React Native for cross-platform support.\n\n## Progress Timeline\n\n- **Phase 1**: Research and Development (Completed)\n- **Phase 2**: Prototype Design (In Progress)\n- **Phase 3**: User Testing and Feedback (Upcoming)\n\n## Future Work\n\nThis project will eventually incorporate AI to learn user preferences and behaviors, making seating personalized and more comfortable.\n\nThis page will be updated as the project continues to progress.
